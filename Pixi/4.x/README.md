## How to build
* $ `npm install`
* $ `npm run build`

## Make sure project structure like this:
```
Your project
    |-- libs
        |-- pixi.js.d.ts
    |-- node_modules
        |-- ...
    |-- out
        |-- ...
    |-- src
        |-- ...
    |-- ...
```
